# Taking Notes and Todo 

Bu uygulama Flutter Frameworku ile yapılmıştır.Uygulamada MVVM yapısı kullanılmıştır.

# Özellikler

- [x] Merkezi Routing
- [x] Provider
- [x] erkezi Tema Yönetimi



Back-end tarafında ise .Net Core-C#-MSSQL-Entity Framework kullanılmıştır.
